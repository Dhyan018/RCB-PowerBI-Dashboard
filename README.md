# 🏏 Royal Challengers Bangalore Performance Dashboard

## 📊 Project Overview

This project presents an interactive Power BI dashboard analyzing the performance of Royal Challengers Bangalore (RCB) across multiple IPL seasons.

## 🎯 Objectives

* Analyze win percentage trends over seasons
* Identify top performers (batsmen & bowlers)
* Compare wins vs losses
* Track overall team performance metrics

## 🛠 Tools & Technologies

* Microsoft Power BI
* SQL (for data querying)
* DAX (for calculated metrics)

## 📈 Key Insights

* Average Win %: 68.80
* Total Matches: 16
* Total Wins: 11
* Top Scorer: Virat Kohli
* Top Wicket-Taker: Yuzvendra Chahal
